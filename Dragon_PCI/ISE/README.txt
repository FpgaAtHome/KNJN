Here are the ISE projects used for KNJN Linux development. 

myeth	 Ethernet/UDP test
myled	 Very simple blinking LED
mypci	 PCI design used with Linux drivers (updated block RAM size)
myusb	 USB IO8
usblcd	 USB text LCD
usbregs	 USB reg banks
binaries Generated .bit files
make	 Makefile replacement to ISE GUI

You should use ISE 10.1.03. Installation should be done with the following 
steps.

1- Get ISE 10.1 webpack from :

http://www.origin.xilinx.com/support/download/index.html/content/xilinx/en/downloadNav/design-tools/v10_1.html

=> ISE Foundation WebPACK - 10.1 Full Product Installation

2- Extract "webpack_SFD.tar", then type "setup" in the "webpack" directory

3- Get SP3 (update) from :

http://www.origin.xilinx.com/support/download/index.html/content/xilinx/en/downloadNav/design-tools/v10_1-sp3.html

4- Extract "10_1_03_lin.zip" then type "setup" in the "10_1_03_lin" directory

You should enter <Xilinx_PATH>/10.1/ISE as directory to update.


Enjoy.
